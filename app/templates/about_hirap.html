About UAV-HiRAP

A guidebook about this platform.

Brief introduction

September 6, 2017 by WANG Haozhou

Unmanned Aerial Vehicles - High Resolution imagery Analysis Platform (UAV-HiRAP), is an open-source, web-based platform which provides service for image classification. Artificial intelligence (AI) have been applied into this platform for better classifing quality.

基于无人机获取的高精度影像和机器学习的人工智能算法，解决各类型植被分类、面积覆盖度提取，林学、生态学等研究领域关键特征的快速计算和输出等需求。

New features

September 7, 2017 by WANG Haozhou

Image Management

The software provides convenient image management function, the user can use project management page free to add, delete items. Also modifing the project name, comments and dealing with the image have been supported.

本软件提供便利的影像管理功能，用户可以用户项目管理页面里自由的添加、删除项目，修改项目名称、备注和欲处理的图像

Decision Tree Model Ggeneration

The classification of the software is based on the artificial intelligence algorithm. The classification model is built automatically by the computer and the basic training set provided by the user. Users can directly use the database has been classified training set, can also use the custom training set.

该软件的分类依据是依赖基于机器学习的人工智能算法进行分类，分类模型由用户提供的分类基础训练集由计算机自动构建。用户可直接利用数据库中已有的分类训练集，也可利用自定义的训练集。

Results Output

After the image classification, results can be outputed in the image tiff format. All types of vegetation coverage can alse be calculated.

在图像分类之后，能将分类后的图片tiff的格式输出，各类型植被覆盖度也能被计算出来。

Sign Up and Sign In

When new user visit, the "Home" and "My Projects" will show please Sign in to unlock UAV-HiRAP vip function.

新用户登陆时，“首页”和“我的项目”均提示需要登录方可解锁功能。

If you did not sign up before, please click "Click here to Sign up" to enter regist page.

如果之前没有注册过账号，则点击Click here to Sign up进入注册界面

Fill in all the information that required, you will get validate email, click the link in that email to activate your account.

填上注册用的信息，将会收到验证邮件，点击邮件里面的链接即可激活邮箱账号。

Back to "Sub Content"

New Project

After signing in, you could enter "New Project" page by clicking "My Projects". Then input project name and select photo, and press "Upload and Classify" button. This step will be automatically finished after web page has response.

登陆后，通过点击My projects，即可进入新建项目界面，键入项目名和选择文件路径之后，点击【Upload and Classify】，等待网页响应后，即完成项目新建与上传。
    <

After all images files are uplaoded, the brower will go into the project content page. The calculation button will be activated.

图片文件上传完毕，直接进入该项目网页中，可以进行计算操作。

Back to "Sub Content"

Open Project

If you have an exist project file, then you can directly open the view. Go to the bottom of this page, click on the hyperlink of project file name 

如果有已经做好的项目文件，那么可以直接打开查看，将网页拉到最下面，点击相应的项目文件名超链接

Then you can go into the project content page.

即可进入项目界面。

Back to "Sub Content"

Modity Project

If you want to modify the project name, project comments and project pictures, click the blue "Edit" button at the bottom of the item.

若想修改项目名称、项目备注和项目图片，点击项目下方蓝色的"Edit"按钮

So you can go into the project modify page.

即可进入项目修改界面。

Users can directly modify the project name and comments in that page, re-upload the picture will replace the original picture.

用户可以直接再网页中修改项目名及备注，重新上传图片后会将原始的图片替换掉。

Back to "Sub Content"

Delete Project

To delete an item, click the gray button at the bottom of the item to delete it.

若想删除项目，点击项目下方灰色的Delete按钮，即可删除改项目

Note: this operation could not be revoked.

注意：此删除操作一旦执行，不可恢复！

Back to "Sub Content"

Image Classification

In the project interface, click the "Calculate" button.

在项目界面中，点击Calculate按钮

The server background will begin to classify images, waiting for the page response, you can get the classification result in the form of binary image.

服务器后台就开始进行分类操作，等待网页响应后，即可获得分类后的二值图

Users could download this binary image by clicking "Download result" button.

用户可以点击"Download result"按钮，下载该黑白二值图。

Back to "Sub Content"

Training pictures generate

September 4, 2017 by JI Jie

Note: Photoshop CS6 is required in this step

  1. Open Photoshop 
    打开Photoshop
  2. New images (Ctrl+N) and set "Background contents" transparent. 
    新建图像（Ctrl+N），设置背景内容为"透明"

  3. Open classified pictures (Ctrl+O) 
    打开要分类的图片（Ctrl+O）

  4. Using Pen tool to select areas where are foreground and background. 
    用钢笔工具来选择背景与前景的训练集

  5. Right click the start point, and select "Make Selection..." 
    在起始点右键，选择"建立选区"

  6. Set "Feather Radius" equal to 0 and press "OK" 
    设置羽化半径为0，点击确定

  7. Copy the select area (Ctrl+C) 
    复制选中的选区（Ctrl+C）

  8. Paste the select area (Ctrl+V) 
    粘贴选中的区域（Ctrl+V）

  9. Repeat previous steps to select enough training picture areas. 
    重复之前的步骤，选取足够的训练集
  10. File > Save As ... 
    文件 > 另存为...

  11. Change the saved format to PNG. 
    修改保存的格式为PNG格式
